# Hrm Frontend
试剂管理系统前端


## 运行环境
Node 14.x  
Node Module 83


## Getting started
```bash
# clone the project
git clone https://gitee.com/journary/hrm.git

# enter the project directory
cd frontend

# install dependency
npm install

# develop
npm start
```
